
# factor

  factor numbers

## Installation

    $ component install component/factor

## Example

```js
factor(4) // [2, 2]
factor(15) // [3, 5]
factor(25) // [5, 5]
factor(27) // [3, 3, 3]
```

## API

### factor(n)

Factor `n` into prime numbers.
